ScrollSticky jQuery Plguin

Sticky objects into the page before they disappear when scrolling the page!

What does this plugin do?

ScrollSticky listen to the scroll event to fix targets position sticky them into the viewport.
You can set a variety of params to control each item behavior.

This is a very useful solution to fix your main menu before it go out of the viewport!
