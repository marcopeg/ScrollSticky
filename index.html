ScrollStiky jQuery Plguin

Stiky objects into the page before they disappear when scrolling the page!

What does this plugin do?

ScrollStiky listen to the scroll event to change position attribute of interested objects to stiky them into the viewport.
You can set a variety of params to control each item behavior.

This is a usefull solution to fix the main menu before it go out of the viewport!

How do I use this demo?

Just scroll down this page and look at the bubble beehavior!

ScrollStiky can be used on every HTML element!

ScrollStiky allow a full automated setup!

ScrollStiky is incredible easy to use!

Even you can configure a variety of params to change dafault behavior!

You can use it for stiky menus or just for fun!
