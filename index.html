ScrollSticky jQuery Plguin

Sticky objects into the page before they disappear when scrolling the page!

What does this plugin do?

ScrollSticky prevent an item to scroll out of the top side of a page.

<div data-scrollsticky="true">stiky element</div>

>> Start to scroll down this page and look at balloons behavior!

ScrollSticky can be used on every HTML element!

ScrollSticky allow a full automated setup!

ScrollSticky is incredible easy to use!

Even you can configure a variety of params to change default behavior!

You can use it for sticky menus or just for fun!

Enjoy it on GitHub!!

 

PushUp Behavior!

ScrollSticky is able to push up sticky items when certain element comes into the page!

<div data-scrollsticky="true" data-scrollsticky-clearfix="#swipe">stiky element</div>

PushUp item is identified by an XPath rule (like CSS).

You can also use an INT value to use a distance as clearfix.
Integer value measure distance from the bottom side of the page.
